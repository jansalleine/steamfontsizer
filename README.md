# steamfontsizer - Version 1.0

## Usage:
>  steamfontsizer [OPTION(s)] [FILE]

## Options:
>  -a NUMBER       add NUMBER to font size__
>                  (default value: 2)__
>  -h, --help      display this helpscreen__
>  -o FILE         specify output file__
>                  (default stdout)__
>  -v              verbose mode__
